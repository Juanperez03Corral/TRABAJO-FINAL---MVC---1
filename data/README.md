# Datos — Instrucciones de descarga

## Descarga del Dataset

1. Accede a la página del INE:  
   https://www.ine.es/dyngs/INEbase/operacion.htm?c=Estadistica_C&cid=1254736176991&menu=resultados&idp=1254735976597#_tabs-1254736195339

2. Descarga los siguientes ficheros:
   - `EILU_GRAD_2019.csv` - microdatos de estudiantes de grado (32.000 registros, 299 variables)
   - `dr_EILU_GRAD_2019.xlsx` - diccionario de variables (necesario para la interpretación)

3. Coloca el fichero `EILU_GRAD_2019.csv` en esta carpeta (`data/raw`).

## Notas de Carga

- **Separador:** (`\t`)
- **Valores perdidos:** celdas vacías y espacios en blanco
